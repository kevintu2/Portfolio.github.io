Recipe App

About

This is a React application that uses a recipe API from Edamam to search for recipes based on what the user decides to search for in the search bar. Once the user picks a key word to search for the app will return a pages full of recipes. Each recipe contains a list of ingredients and an estimated calorie count for the dish. This is built with React.js, CSS, and Javascript
