Bootleg Facebook

About

I created a simple social media app by utilizing multi-threaded programs, multicast communication, and application-layer protocols. This app consists of one server and multiple clients. The clients are able to connect to the server and able to interact with people within the server. Users are able to post statuses, but only their friends can see their statuses. Users also have the ability to send friend requests, accept requests, deny requests, and unfriend someone. Once two users become friends they can see each other's statuses. While new users join the server, everyone will recieve a broadcast of who is joining. Same goes for whoever leaves the server.

This was all built with Java and is able to be deployed on linux clusters.
